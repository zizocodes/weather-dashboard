# 🌦 Weather Dashboard App

A modern interactive weather dashboard built using HTML, CSS, and JavaScript.

## Features

- Live weather data using OpenWeather API
- City search functionality
- Auto-detect user location weather
- 5-day forecast section
- Dynamic background based on weather conditions
- Animated charts using Chart.js
- Dark / Light mode toggle
- Recent search history (localStorage)
- Fully responsive design

## Tech Stack

HTML
CSS
JavaScript
Chart.js
OpenWeather API

## Setup Instructions

1. Clone repository

git clone https://github.com/YOUR_USERNAME/weather-dashboard-app.git

2. Open script.js

Add your OpenWeather API key:

const API_KEY = "YOUR_API_KEY"

3. Open index.html in browser
